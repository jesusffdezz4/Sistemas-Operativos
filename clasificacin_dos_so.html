Clasificación de los SO

Los sistemas operativos se pueden clasificar según diferentes criterios:

Modo de operación

  • Procesamiento por lotes : Este modo apareció en la segunda generación y consiste en realizar varias tareas secuencialmente, es decir, hasta que termine la primera, la segunda no comienza, etc. Por lo tanto, no se realizan varias tareas simultáneamente. Los tiempos de respuesta suelen ser elevados.

Nota: el tiempo de respuesta es el tiempo transcurrido entre cuando se ordena ejecutar un programa y el momento en que devuelve un resultado.

  • Timeshares o interactivo : El sistema es capaz de simular que ejecuta simultáneamente varias tareas, pero lo que hace es alternar pequeñas partes de una y otra, intentando optimizar los tiempos de respuesta de todas ellas. Es común, por ejemplo, dar preferencia a aquellos procesos cuya resolución impacienta al usuario, como responder a las pulsaciones de teclas, y dejar en segundo plano otros procesos a los que el usuario no presta atención.

Los sistemas operativos genéricos modernos utilizan este modo de funcionamiento, sin perjuicio de que a veces les interese aplicar el procesamiento por lotes.

  • Tiempo real : Es el modo en que un tiempo de respuesta fuera del límite predeterminado, que generalmente es muy corto, es inaceptable y provoca fallos. Los sistemas que operan con este modo de operación se usan normalmente en áreas muy específicas como el control del tráfico aéreo, los sistemas de conducción automática o los sistemas de fabricación automática industrial.

Número de usuarios

Dependiendo del número de usuarios que pueden usar el sistema simultáneamente:

  • Una única sesión de usuario a la vez o usuario único: Son aquellos en los que solo un usuario puede usar el sistema a la vez. Puede ser un único usuario estricto o usuario único en interacción, que permiten dejar los programas de otros usuarios en segundo plano, pero no permiten la interacción simultánea de dos usuarios diferentes.

Aviso: un sistema de usuario único no es exclusivamente un sistema de usuario único

  • Multiusuario : Son aquellos en los que varios usuarios pueden interactuar simultáneamente con el sistema operativo y consumir recursos. Todos los derivados de UNIX (Linux, FreeBSD, MacOs v10.7, también llamados Lion) y las versiones de servidor de Windows (NT, 20XX) son sistemas multiusuario.

Número de procesos

Un proceso es un programa en ejecución cuyas instrucciones están cargadas total o parcialmente en la memoria principal . Por lo tanto, un programa, antes de ejecutarlo, no es más que un archivo o un conjunto de archivos almacenados en la memoria secundaria (normalmente en el disco duro). En cuanto se ordena su ejecución, se carga en la memoria y se constituye un proceso. Cada ejecución, por otro lado, constituye un proceso diferente.

Este criterio aborda el número de procesos que el sistema operativo es capaz de gestionar simultáneamente:

  • Tarea única (monoprogramación) : Son los sistemas operativos que permiten la ejecución de un único proceso simultáneamente, de modo que no se puede ejecutar un segundo proceso hasta que el primero no termine.

Ya no se utilizan sistemas operativos de tarea única. En el pasado, MS-DOS era una tarea única.

  • Multitarea (o multiprogramación) : Son sistemas en los que es posible la ejecución simultánea de varios procesos. Esto solo es posible cuando el sistema tiene varios procesadores, pero en los sistemas con un solo procesador, se utiliza la técnica de tiempo compartido para emular la simultaneidad de los procesos. La multitarea requiere un algoritmo de programación de procesos que se verá cuando se discuta la gestión de procesos. Los sistemas operativos modernos son multitarea.

Número de procesadores

Relacionado con el número de procesadores disponibles en el sistema informático:

  • Proceso único : Son sistemas operativos que solo son capaces de usar un procesador. Sin embargo, esto no significa que sean necesariamente de tarea única, ya que pueden usar la técnica de tiempo compartido para realizar varias tareas. Son sistemas operativos MS-DOS de proceso único y sistemas Windows fuera de la rama NT (Windows 9X, Windows Me).
  • Multiproceso : Son sistemas operativos capaces de usar simultáneamente varios procesadores. En consecuencia, estos sistemas operativos ejecutan simultáneamente varias instrucciones, una en cada procesador, sin que esto sea un obstáculo para utilizar técnicas como la multipropiedad.
